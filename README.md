# diszkertek.hu – statikus új verzió

A diszkertek.hu teljesen új, statikus HTML/CSS/JS alapú változata.  
Cél: hipergyors betöltés, optimalizált képek, egyszerű karbantartás, többoldalas felépítés.

## Mappastruktúra

/diszkertek  
│  
├── index.html  
├── szolgaltatasok.html  
├── referenciak.html  
├── rolunk.html  
├── kapcsolat.html  
│  
└── assets  
    ├── css  
    │   └── style.css  
    ├── js  
    │   └── script.js  
    ├── img  
    │   └── (képek)  
    └── fonts (opcionális)

## Technológia

- HTML5  
- CSS3 (Flexbox, Grid, reszponzív töréspontok)  
- JavaScript (minimális interakciók, mobilmenü)

## Futtatás

Nyisd meg az `index.html` fájlt böngészőben.

A projekt minden eleme szerzői jogvédelem alatt áll.
